# 学習時間記録アプリ

学習時間を記録・管理できるWebアプリです。
日別・週別・月別・年別・総合計で学習時間を確認できます。

---

## 概要

このアプリは以下の目的で作成しました。

- 学習時間を簡単に記録できるようにする
- 学習量を可視化して継続しやすくする
- 実務を想定したCRUD・認可・バリデーションの実装

---

## 主な機能

### 学習記録

- 学習時間の登録（30分単位）
- 学習記録の編集（モーダル）
- 学習記録の削除
- 日付ごとの記録表示

### カテゴリー

- カテゴリーの登録
- カテゴリーの編集
- カテゴリーの削除

### 集計

- 日別合計
- 週別合計
- 月別合計
- 年別合計
- 総合計

セレクトボックスで表示切替可能

---

## 技術スタック

- バックエンド：Laravel
- フロントエンド：React + Inertia.js
- スタイリング：TailwindCSS
- データベース：SQLite（開発環境）
